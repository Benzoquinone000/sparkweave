# 学习画像 P18：把推进风格变成用户看得懂的卡片

这一轮继续围绕“画像是重中之重”推进，但不增加复杂配置。

## 问题

画像页已经能推断“练习驱动型、概念澄清型、快速串联型”等推进风格，也能计算最近证据是否稳定。但这些信息如果只藏在标签或底层逻辑里，普通用户不会感知到：

> 系统到底准备怎么教我？

## 本轮目标

在画像概览中增加一张轻量卡片，让用户一眼看见：

1. 自己当前更像哪种学习推进方式。
2. 系统为什么这样判断。
3. 最近证据是在变稳、提速、修正，还是继续观察。
4. 接下来系统会按什么策略安排导学和资源。

## 本轮实现

前端 `/memory` 概览页新增“你的学习推进方式”卡片：

- 显示当前推进风格标签。
- 用一句自然语言解释学习节奏。
- 展示三个支撑信号：起步方式、稳定程度、反馈习惯。
- 展示最近趋势：变稳、提速、修正或观察。
- 保留一个进入导学的轻入口，避免用户看完画像后不知道下一步去哪。

## 价值

这一层让画像从“系统对用户的判断”变成“用户可理解的学习协定”：

- 对用户：知道系统接下来为什么这样安排。
- 对导学：能把推荐资源和学习节奏说得更像个性化辅导。
- 对比赛：更直观地展示“对话式学习画像自主构建”和“个性化路径动态调整”。
