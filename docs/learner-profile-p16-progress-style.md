# 学习画像 P16：学习推进风格画像

这一轮把“学习效果评估”和“画像中心”再往前接了一层。

## 问题

前面的画像页已经能回答：

- 你现在重点学什么
- 你哪里薄弱
- 你掌握得怎么样
- 最近发生了什么变化

但它还不太回答另一个很重要的问题：

> 你通常是怎么把一个知识点学会的？

这其实很适合比赛语境里的“个性化学习画像”。

## 本轮目标

让画像页不仅描述“状态”，也开始描述“推进方式”。

## 本轮实现

新增了一个“学习推进风格”板块，会基于当前已有画像数据进行轻推断：

- 偏好
- 掌握度均值
- 证据强度
- 薄弱点数量
- 校准次数
- 当前画像可信度

然后生成更稳定的风格标签，例如：

- 练习驱动型
- 概念澄清型
- 快速串联型
- 反复校准型
- 渐进压实型

## 展示内容

这个板块会展示三类信息：

1. 一个核心风格标签
2. 一段自然语言摘要
3. 三个支撑信号
4. 系统据此更偏向的导学策略

也就是说，它不只是“你是谁”，还会解释“所以系统接下来为什么更倾向这样教你”。

## 价值

这一层很适合用于比赛展示，因为它把画像从“字段集合”往“学习者模型”推进了一步：

- 不只是知道用户的薄弱点
- 还开始知道用户更适合怎么推进
- 同时能把这种判断反哺到导学与资源生成

## 下一步建议

下一步可以继续做两件事中的一件：

1. 把这种“推进风格”同步到导学页，让策略卡更明确地引用它
2. 把这类风格沉淀成长期趋势，在多轮学习后展示“你的风格是否发生了变化”
