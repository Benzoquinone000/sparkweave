# 稳定课程 Demo 模板

SparkWeave 现在内置三条适合比赛演示的课程路线。它们不是普通示例数据，而是为了在 7 分钟视频、PPT 答辩和现场演示中稳定覆盖赛题五条要求而准备的“可复现课程故事”。

## 推荐选择

| 模板 | 适合展示 | 推荐场景 |
| --- | --- | --- |
| 机器学习基础 | 画像、路径、图解、交互练习、学习报告完整闭环 | 最稳的主演示路线 |
| 智能机器人与 ROS 基础 | 项目式课程、代码/机器人知识库、公开视频、图解练习 | 展示“完整高校专业课程” |
| 高等数学极限与导数 | 公式渲染、极限图解、Manim 动画、分层练习 | 展示多模态数学辅导 |

## 机器学习基础

入口：

- `/guide` 点击“开始稳定演示”。

推荐讲法：

1. 从画像进入导学，说明系统先理解学生目标、薄弱点和资源偏好。
2. 展示机器学习课程路线和当前任务。
3. 生成“梯度下降图解”或展示已保存图解。
4. 完成“模型评估混合题”，提交分数和反思。
5. 打开学习报告和课程产出包，说明画像、资源和反馈已经成闭环。

稳定任务链：

- `T1`：建立机器学习全景图。
- `T4`：画出梯度下降过程。
- `T6`：从混淆矩阵计算指标。

兜底材料：

- 图解：梯度下降损失曲线、负梯度方向和学习率影响。
- 练习：准确率、精确率、召回率、F1。
- 话术：如果生成慢，直接展示课程产出包里的稳定提示词和历史产物。

## 智能机器人与 ROS 基础

入口：

- `/guide` 点击“智能机器人与 ROS 基础”课程卡片，再点击“帮我安排学习”。

推荐讲法：

1. 说明这是一门 8 周项目式课程，最终目标是移动机器人通信与控制小项目。
2. 展示机器人系统全景图：感知、决策、控制、执行。
3. 生成 ROS publisher/subscriber 通信图，说明话题消息如何流动。
4. 展示 TF 坐标树或 cmd_vel 控制闭环图解。
5. 打开课程产出包，把节点架构、调试证据和项目报告作为最终成果。

稳定任务链：

- `R1`：画出机器人系统全景图。
- `R3`：实现话题发布与订阅。
- `R5`：画出 TF 坐标树。
- `R6`：解释 cmd_vel 控制闭环。

兜底材料：

- 图解：机器人系统全景图、ROS 通信图、TF 坐标树。
- 练习：话题、服务、参数适用场景判断题。
- 公开视频：用于补充真实机器人系统案例。
- 话术：这条路线强调“课程项目”，不是单点问答。

## 高等数学极限与导数

入口：

- `/guide` 点击“高等数学极限与导数”课程卡片，再点击“帮我安排学习”。

推荐讲法：

1. 说明这是一门 6 周微积分入门课程，目标是把公式、图像和练习串起来。
2. 先做左右极限判断，展示交互式题目和反馈。
3. 生成极限图解，解释“趋近但不一定取到”。
4. 生成或展示割线逼近切线的 Manim 动画，说明导数的几何意义。
5. 用链式法则练习回写画像，再打开学习报告看下一步处方。

稳定任务链：

- `M2`：判断左右极限是否存在。
- `M4`：生成割线逼近切线动画。
- `M5`：链式法则分层练习。

兜底材料：

- 图解：左右极限、连续性、函数局部变化。
- 动画：割线斜率逐步逼近切线斜率。
- 练习：链式法则选择、填空、简答。
- 话术：这条路线最适合突出“多模态数学辅导”加分项。

## 如何扩展新课程

新增课程模板放在：

```text
data/course_templates/<template_id>.json
```

推荐至少包含：

- `nodes`：课程知识点。
- `tasks`：可执行学习任务。
- `learning_path`：节点顺序和今日重点。
- `recommendations`：资源和练习建议。
- `demo_seed`：录屏稳定材料。

`demo_seed` 最小结构：

```json
{
  "title": "稳定演示样例",
  "scenario": "从画像到资源再到反馈的演示故事。",
  "persona": {
    "goal": "学习目标",
    "weak_points": ["主要卡点"],
    "preferences": ["visual", "practice"]
  },
  "task_chain": ["T1", "T2"],
  "resource_prompts": {
    "T1": "生成一张概念图。",
    "T2": "生成一组交互练习。"
  }
}
```

提交前运行：

```powershell
python scripts/check_course_templates.py
python -m pytest tests/services/test_guide_v2.py -q
```
