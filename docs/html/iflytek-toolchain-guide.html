科大讯飞工具链说明

赛题要求作品选用科大讯飞相关工具。SparkWeave 的处理方式不是把讯飞能力单独做成展示菜单，而是把它们放进学生学习过程：资料先能被读懂，问题先能找到依据，图片和公式能转成可回答内容，讲解可以变成语音，学习结果能回到记录里。

评委可以从 设置页 查看配置入口，也可以在学习、资料、问问题和记录页面观察这些能力如何参与学习流程。

学习流程中的讯飞能力

学习环节 使用的讯飞能力 对学生的价值 页面入口
课程资料入库 Spark Embedding、OCR for LLM 让 PDF、扫描讲义和图片资料进入资料库 资料页
资料问答 星火大模型、Spark Embedding、ONE SEARCH 回答课程问题时先找资料来源，再补充公开资料 资料页、问问题页
图片和公式题 公式识别、图片理解、OCR 题图、板书、截图先被转成可解释内容 问问题页、图像解题
学习资源生成 星火大模型、MaaS Coding、星辰工作流 生成图解、练习、讲解材料和课程成果 学习页、问问题页
语音讲解 超拟人 TTS、语音听写 ASR 支持口述提问、音频讲解和短视频旁白 问问题页、学习页
学习效果记录 语音评测、练习反馈、学习报告 把口语练习、作答结果和反馈写入记录 记录页

这张表可以放进“讯飞工具链”相关材料。它比单纯列 API 名更容易说明作品怎么服务学习。

具体落点

讯飞能力 SparkWeave 中的落点 核验方式
星火大模型 问答模型配置 作为主要对话和资源生成模型，回答课程问题
MaaS Coding / Astron Code 代码模型配置 用于代码类课程任务、工具调用和代码讲解
Spark Embedding 向量模型配置 上传课程资料后，使用向量检索支撑资料问答
ONE SEARCH 搜索服务配置 当课程资料不足时，补充公开资料和视频线索
OCR for LLM OCR 配置 扫描课件、图片讲义或图片资料入库
公式识别 iflytek_formula_ocr 数学题图和手写公式进入解题流程
图片理解 iflytek_image_understanding 板书、实验图、截图进入多模态答疑
超拟人 TTS 语音合成配置 为讲解材料生成音频或短视频旁白
语音听写 ASR 语音识别配置 学生用口述方式提问或记录反思
语音评测 ISE 语音评测配置 口语、朗读或表达类任务形成学习证据
星辰工作流 iflytek_workflow 接入已发布工作流，生成课程资源或诊断报告

这些能力都能在设置页找到配置或检测入口。不需要逐个点开所有服务，建议固定两到三项真实可用服务，把流程走顺。

推荐核验路线

讯飞工具可以这样穿插在学习流程中，不需要单独占满一大段时间：

页面 操作 核验重点
资料页 上传一份课程资料或展示已有资料库 OCR、Embedding 与资料问答有关
问问题页 围绕资料提问 回答旁能看到资料来源
问问题页 上传公式题或截图 公式识别、图片理解把图片转成可辅导内容
学习页 生成图解、练习或讲解材料 资源不是随便生成，而是围绕当前课程任务
记录页 查看学习记录和下一步建议 学习结果会影响后续推荐
设置页 展示讯飞配置和连接检测 说明哪些服务已真实配置，哪些仅用于流程验证

更稳的讲法是：先展示一项真实服务调用，再展示学习流程。这样既能看到工具接入，也能看到它确实服务学习任务。

最小可展示组合

如果账号权限或现场网络有限，建议优先保证下面这组能力：

优先级 能力 原因
1 星火大模型 支撑对话式辅导、资源生成和学习路径解释
2 Spark Embedding 支撑课程资料向量化和资料问答
3 ONE SEARCH 支撑公开资料补充和视频资源建议
4 OCR 或公式识别 展示多模态输入，尤其适合数学和扫描讲义
5 TTS 展示多模态输出，用于语音讲解或短视频旁白

这组能力已经能覆盖赛题中的资源生成、智能辅导、资料证据和多模态需求。ASR、语音评测和星辰工作流可以作为补充能力。

页面上如何说明真实性

提交和核验时，建议把服务状态说清楚：

情况 建议说法
已配置真实密钥 “这里是通过设置页检测通过的讯飞服务，当前结果来自真实调用。”
暂时没有权限 “这里展示的是系统流程，页面会标明不是一次真实云端返回。”
网络不稳定 “系统保留已生成材料，真实调用入口仍在设置页和工具层。”
不展示完整密钥 “密钥只放在本地 .env，录屏、截图和提交包不会出现真实凭据。”

把真实调用、替代结果和密钥边界说清楚，比含糊带过更可靠。

核验动作

如果需要核验，可以按下面顺序看：

顺序 页面 核验内容
1 设置页 查看星火、Embedding、搜索、OCR、语音和工作流配置入口
2 设置页 运行连接检测，确认已配置服务的检测结果
3 资料页 上传或选择课程资料，确认资料库能用于问答
4 问问题页 围绕资料提问，展开资料来源
5 问问题页 使用公式识别、图片理解或 TTS 相关能力生成一次多模态结果
6 记录页 查看学习记录和下一步建议是否更新

这条路线能同时看到讯飞工具、资料证据、智能辅导和学习记录，不需要在页面之间来回找功能。
